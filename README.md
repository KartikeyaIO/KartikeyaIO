# 💫 About Me:
Building **Editron** — a media processing DSL focused on performance, control, and composable pipeline design.

Designed to abstract media workflows while retaining low-level control over execution.

Focused on systems programming, embedded systems, and performance-critical design.

I work close to the hardware, building efficient pipelines and exploring how high-level abstractions translate into real-world performance.

---

## 🚀 Featured Project

### 🔧 Editron
A media processing DSL designed for control and performance.

- Custom pipeline abstraction for frame-based processing  
- Uses FFmpeg for encoding/decoding as a backend  
- Focus on performance, composability, and control  
- Built with Rust
